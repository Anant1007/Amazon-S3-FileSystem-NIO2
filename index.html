Amazon S3 API JSR-203 for Java 7

An Amazon AWS S3 FileSystem Provider for Java 7 (NIO2)

Download .zip Download .tar.gz View on GitHub

Amazon Simple Storage Service (S3) provides a fully redundant data storage infrastructure for storing and retrieving any amount of data, at any time. JSR-203, NIO2, is the new file management API, introduced in Java version 7. This project provides a first API implementation, little optimized, but "complete" to manage files and folders directly on Amazon S3.

Features:

  • Copy and create folders and files
  • Delete folders and files
  • Copy paths between different providers
  • Walk file tree
  • Works with virtual s3 folders (not really exists and are element's subkeys)

Roadmap:

  • Performance issue (slow querys with virtual folders)
  • Better test coverag)

  • Multi endpoint fileSystem (Actually one FileSystem at the same time)
